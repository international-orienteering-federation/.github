Welcome to the github home of the International Orienteering Federation. This repo is primarily maintained by member of the IOF IT Commission.

The background of the official [IOF Data Standard](https://orienteering.sport/iof/it/data-standard-3-0/) can be found on our [homepage](https://orienteering.sport/iof/it/data-standard-3-0/), 
but the actually details, and work on the next versions are here on github.

We maintain the requirements for [Electronic Punching in orienteering](https://orienteering.sport/iof/it/electronic-punching/), 
with an [approval process](https://orienteering.sport/wp-content/uploads/2026/03/iof-punching-system-approval-v-2018-04-07-3.pdf) 
and a [list of approved systems.](https://orienteering.sport/wp-content/uploads/2026/04/iof-approved-electronic-punching-systems-april-2026.pdf)
